1. З папки Prefabs дістати  WheelOfLuck.
2. Створити Config/Splin/WheelPiecesSetting.
3. Налаштувати WheelPiecesSetting, заповнивши поля іконки, назви, кількості, шансу випадіння та випадіння перших n нагород(якщо 0 то нагороди використовується одразу по шансу випадіння, наприклад, значення 1 буде першим, 2 - другим і тд., після завершення всіх чисел продовжується по шансу випадання) 
Pieces-це ті поля які будуть одразу відресовані і така кількість буде на рулетці,
AdditionalPieces - додаткові поля які будуть підставлятись після отримання нагороди. Можна рандомно вибрати(встановши галочку UseRandomSupportPiecesList) модна по пріоритету виставити аналогічно з попереднім. Якщо AdditionalPieces пустий то замінятись предмети не будуть.
4. Перетягнути створений WheelPiecesSetting в відповідне поле в префабі WheelOfLuck.
5. Створити кнопку і викликати метод Spin в PickerWheel.
